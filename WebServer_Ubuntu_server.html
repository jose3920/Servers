1. Instalamos apache con apt-get install apache2
  2. Configuramos IP con netplan
  3. Reiniciamos servicio
  4. Creamos directorio dentro de /var/www/*directotio*
  5. Creamos HTML dentro del directorio
  6. Dentro de /etc/apache2/sites-available copiamos el archivo 000-default.conf y cambiamos su nombre
  7. Editamos y actualizamos la informacion
  8. ejecutamos el a2ensite archivo.conf y comprobamos que en /etc/apache2/sites-enabled se ha creado el archivo
  9. Reiniciamos apache y todo funcionará correctamente.
  10. Creamos HTML dentro del directorio

    11. Nota: Para cambiar puertos de escucha acceder a /etc/apache2/ports.conf y añadir los puertos
